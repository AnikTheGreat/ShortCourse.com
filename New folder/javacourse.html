About this Course

This course aims to teach everyone the basics of programming computers using JAVA. We cover the basics of how one constructs a program from a series of simple instructions in JAVA. The course has no pre-requisites and avoids all but the simplest mathematics. Anyone with moderate computer experience should be able to master the materials in this course. This course will cover Chapters 1-5 of the textbook “JAVA for Everybody”. Once a student completes this course, they will be ready to take more advanced programming courses. This course covers Python 3.

Syllabus - What you will learn from this course

Chapter One: Variables and Data types in JAVA

In this chapter we cover how a program uses the computer's memory to store, retrieve and calculate information.

Approximately 2 hours to complete

Chapter Two: Conditional Statements and Switch Case

In this section we move from sequential code that simply runs one line of code after another to conditional code where some steps are skipped. It is a very simple concept - but it is how computer software makes "choices".

Approximately 2 hours to complete

Chapter Three: Loops and Iteration

Loops and iteration complete our four basic programming patterns. Loops are the way we tell JAVA to do something over and over. Loops are the way we build programs that stay with a problem until the problem is solved.

Approximately 2 hours to complete

Chapter Four: Functions

This is a relatively short chapter. We will learn about what functions are and how we can use them. The programs in the first chapters of the book are not large enough to require us to develop functions, but as the book moves into more and more complex programs, functions will be an essential way for us to make sense of our code.

Approximately 2 hours to complete
